Message as a single image



This is just for fun. The entire message is embedded in a single image, with each pixel corresponding to one symbol in the message. The image reads diagonally, starting at the top left. It is handy for spotting phase changes in the message (e.g. the transition from using unary to binary, or - more importantly - bugs). Zeros are greenish, ones are reddish, parentheses (symbols 2 and 3) are different shades of blue.
